    HTML (Hypertext Markup Language) is used for creating and structuring content for the web. Some of the main uses of HTML include:

  • Creating the structure of a webpage: HTML is used to define the basic structure of a webpage, including headings, paragraphs, lists, and other content elements.
  • Formatting text and other content: HTML is used to format text and other content elements using styles, such as font size, color, and alignment.
  • Adding images and multimedia: HTML is used to embed images, videos, and audio files into webpages, allowing them to be viewed and played by website visitors.
  • Creating hyperlinks: HTML is used to create hyperlinks that allow website visitors to navigate between pages and websites.
  • Building web forms: HTML is used to create web forms that allow website visitors to input and submit data, such as contact information, feedback, and survey responses.
  • Search engine optimization: HTML is used to structure content on a webpage in a way that makes it more accessible to search engines, helping to improve the page's visibility and ranking in search results
  • Mobile optimization: HTML is used to create responsive websites that can adapt to different screen sizes and devices, providing an optimal viewing experience for all visitors.

    CSS (Cascading Style Sheets) is used to control the presentation and styling of web content. It is used in conjunction with HTML to provide a wide range of design and layout options for webpages. Some of the main reasons why CSS is used include:

  • Separation of content and presentation: CSS allows you to separate the content and structure of a webpage from its presentation and styling, making it easier to maintain and update the website in the long run.
  • Consistency and flexibility: CSS allows you to define consistent styles and layouts for multiple webpages on a website, providing a unified and professional look and feel. It also allows for flexibility in changing the presentation of the website, without affecting the underlying content and structure
  • Design and layout options: CSS provides a wide range of design and layout options, including font styles, colors, backgrounds, margins, borders, and more, allowing for greater creativity and customization in webpage design.
  • Accessibility: CSS can be used to optimize the presentation of web content for different devices and screen sizes, improving the accessibility and usability of the website for a wider audience.
  • Search engine optimization: CSS can be used to structure and optimize web content for search engines, helping to improve the website's visibility and ranking in search results.

    JavaScript is used as a client-side scripting language on the web to enhance the functionality and interactivity of webpages. Some of the main reasons why JavaScript is used include:

  • Dynamic content: JavaScript allows you to create dynamic and interactive web content that can be updated and modified in real-time based on user actions, making the website more engaging and responsive.
  • Webpage behavior: JavaScript can be used to control the behavior of webpages, such as validating user inputs in web forms, displaying alerts or pop-ups, and creating interactive animations and effects
  • User interface enhancements: JavaScript can be used to enhance the user interface of webpages, such as creating drop-down menus, sliders, and other interactive widgets that make it easier for users to navigate and interact with the website
  • Web application development: JavaScript can be used to develop complex web applications, such as games, social media platforms, and e-commerce websites, that require advanced client-side functionalit
  • Cross-platform compatibility: JavaScript can be executed on any device with a web browser, making it a versatile and widely used language for web development.
  • Integration with other technologies: JavaScript can be used in conjunction with other web technologies, such as HTML and CSS, as well as various libraries and frameworks, to provide even greater functionality and flexibility for web development.

function verifyPassword(){
var ps=document.getElementById("password").value;
var cps=document.querySelector("#con-password").value;
if(ps.length < 8)
{
    document.getElementById("messege").classList.add("messege");
    document.getElementById("password").value="";
    document.querySelector("#con-password").value="";
    document.getElementById("messege").innerHTML="** Password length must be grater than 8 characters!";
    return false;
}
if(ps.length > 15)
{
    document.getElementById("messege").classList.add("messege");
    document.getElementById("password").value="";
    document.querySelector("#con-password").value="";
    document.getElementById("messege").innerHTML="** Password length must not exceed 15 characters!";

    return false;
}
if(ps != cps)
{
    document.querySelector("#con-messege").classList.add("messege");
    document.querySelector("#con-messege").innerHTML="** Password not matched!";
    document.getElementById("password").value="";
    document.querySelector("#con-password").value="";
    return false;
}
}
